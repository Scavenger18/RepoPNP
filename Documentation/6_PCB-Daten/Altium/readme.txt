Das Sprocket-Projekt wurde in Altium umgesetzt. Das Umsetzten von DXF-Files in PCB Dokumente funktioniert deutlich besser in Altium. 
Für Änderungen am Sprocket in einem anderen Programm kann das DXF-File bei den CAD Daten entnommen werden. 
Sollten nur die Bohrungen (0.6mm) verändert werden, kann in den Project Outputs das Text-File für die NC-Drilldaten verändert werden.
In der Datei Gear_45_Double.txt kann auf der Bohrungsdurchmesser von 24 mil (0.6mm) auf der Linie 

T1F00S00C0.02400

verändert werden. Dabei kann statt 0.02400 ein anderer Durchmesser eingefügt werden.
